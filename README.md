# Welcome to the SDN-Dashboard Frontend

## Project info
React Typescript application with interactive network topology visualization

The Frontend is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Project Sections

The dashboard provides comprehensive visualization and monitoring of Software-Defined Network (SDN) infrastructure through multiple interactive views:

### 1. **Node Details Sidebar**
A persistent sidebar that displays all SDN nodes in the network with real-time status monitoring.

### 2. **Map View**
Visual representation of network topology in a grid-based geographical layout.

### 3. **Topology View**
Hierarchical network visualization showing the SDN architecture layers.

### 4. **Extended Node View**
Detailed information panel for comprehensive node analysis.

### 5. **Messages Window**
Live feed of network communication and system events.

### 6. **Route Analysis**
Path visualization and route status monitoring.

## Project Development

You can work locally using your preffered IDE by clonning this repo and introduce changes by sending a PR.

**Requirements**

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone "<PROJECT_GIT_URL>"

# Step 2: Navigate to the project directory.
cd "meshfyp-network-compass"

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server.
npm run dev
```
